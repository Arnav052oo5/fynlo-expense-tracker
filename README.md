# Expense Tracker CLI

A simple command-line Expense Tracker application built with Python to help users manage personal finances efficiently.

---

## Features

- Add new expenses with description and amount
- Update existing expenses
- Delete expenses
- View all expenses
- View total expense summary
- View monthly expense summary
- JSON-based file storage
- Error handling for invalid inputs
- Simple and beginner-friendly CLI interface

---

## Technologies Used

- Python
- argparse
- JSON
- File Handling
- Datetime
